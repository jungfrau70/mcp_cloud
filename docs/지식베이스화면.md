# 지식베이스 화면 명세서

## 📋 개요

지식베이스 화면은 내부 자료 검색과 외부 자료 기반 AI 문서 생성을 통합한 사용자 인터페이스입니다. 탭 기반 구조로 두 가지 주요 기능을 명확히 구분하여 제공합니다.

## 🎨 화면 구조

### 1. 헤더 영역

#### 1.1 메인 헤더
```
┌─────────────────────────────────────────────────────────────┐
│ 지식베이스                    총 25개 문서                  │
├─────────────────────────────────────────────────────────────┤
│ [내부자료 검색] [외부자료 기반 문서생성]                     │
└─────────────────────────────────────────────────────────────┘
```

**구성 요소:**
- **제목**: "지식베이스" (text-2xl font-bold)
- **문서 수 표시**: "총 X개 문서" (text-sm text-gray-500)
- **탭 네비게이션**: 활성 탭 하이라이팅 (border-blue-500)

#### 1.2 탭 네비게이션
- **내부자료 검색**: 기존 문서 관리 및 검색
- **외부자료 기반 문서생성**: AI 기반 문서 생성
- **활성 상태**: 파란색 하단 보더 및 텍스트 색상 변경

### 3. 외부자료 기반 문서생성 탭

#### 3.1 전체 레이아웃
```
┌─────────────────────────────────────────────────────────┐
│                    📚 아이콘                             │
│              외부자료 기반 문서생성                       │
│                                                         │
│  AI가 외부 웹사이트와 기술 문서를 검색하여 지식베이스    │
│  문서를 자동으로 생성합니다.                             │
│                                                         │
│  ┌─────────────────────────────────────────────────────┐ │
│  │  문서 주제 *                                        │ │
│  │  ┌─────────────────────────────────────────────────┐ │ │
│  │  │ 예: AWS Lambda를 사용한 서버리스 애플리케이션   │ │ │
│  │  │ 배포 방법                                        │ │ │
│  │  └─────────────────────────────────────────────────┘ │ │
│  │                                                     │ │
│  │  문서 타입                                          │ │
│  │  [기술 가이드 ▼]                                    │ │
│  │                                                     │ │
│  │  검색 소스                                          │ │
│  │  ☑ 웹 검색  ☑ 뉴스 검색  ☐ 문서 검색               │ │
│  │                                                     │ │
│  │  저장 경로 (선택사항)                               │ │
│  │  [예: aws/lambda-deployment]                        │ │
│  │                                                     │ │
│  │  최대 검색 결과 수                                  │ │
│  │  [5]                                               │ │
│  │                                                     │ │
│  │  [AI 문서 생성 시작]                                │ │
│  └─────────────────────────────────────────────────────┘ │
└─────────────────────────────────────────────────────────┘
```

#### 3.2 헤더 섹션
- **아이콘**: 전구 아이콘 (SVG)
- **제목**: "외부자료 기반 문서생성" (text-2xl font-bold)
- **설명**: 기능 설명 텍스트 (text-gray-600)

#### 3.3 폼 섹션 (max-w-2xl mx-auto)

##### 문서 주제 입력
```
문서 주제 *
┌─────────────────────────────────────────────────────────┐
│ 예: AWS Lambda를 사용한 서버리스 애플리케이션 배포 방법   │
│                                                         │
└─────────────────────────────────────────────────────────┘
```

**스타일링:**
- **라벨**: `block text-sm font-medium text-gray-700 mb-2`
- **텍스트 영역**: `w-full px-3 py-2 border border-gray-300 rounded-md`
- **플레이스홀더**: 예시 텍스트 제공

##### 문서 타입 선택
```
문서 타입
[기술 가이드 ▼]
```

**옵션:**
- 기술 가이드 (guide)
- 단계별 튜토리얼 (tutorial)
- 참조 문서 (reference)
- 비교 분석 (comparison)

##### 검색 소스 선택
```
검색 소스
☑ 웹 검색    ☑ 뉴스 검색    ☐ 문서 검색
```

**스타일링:**
- **체크박스**: `rounded border-gray-300 text-blue-600 focus:ring-blue-500`
- **라벨**: `ml-2 text-sm text-gray-700`

##### 고급 설정
```

[예: aws/lambda-deployment]
비워두면 루트 디렉토리에 저장됩니다

최대 검색 결과 수
[5]
```

**스타일링:**
- **입력창**: `w-full px-3 py-2 border border-gray-300 rounded-md`
- **도움말 텍스트**: `mt-1 text-xs text-gray-500`

##### 생성 버튼
```

```

**상태별 스타일링:**
- **기본**: `w-full px-4 py-3 bg-blue-600 text-white rounded-md`
- **호버**: `hover:bg-blue-700`
- **비활성**: `disabled:opacity-50 disabled:cursor-not-allowed`
- **로딩**: 스피너 아이콘 + "문서 생성 중..." 텍스트

## 🔧 인터랙션

### 탭 전환
- **클릭 이벤트**: `@click="activeTab = 'internal|external'"`
- **활성 상태**: `activeTab === 'internal|external'`
- **스타일 변경**: 보더 색상 및 텍스트 색상 변경

### 폼 검증
- **실시간 검증**: `computed(() => externalQuery.value.trim().length > 0)`
- **버튼 비활성화**: `:disabled="!canGenerateExternal"`
- **시각적 피드백**: 버튼 투명도 및 커서 변경

### 로딩 상태
- **스피너 애니메이션**: `animate-spin`
- **진행률 업데이트**: `externalProgress.value = 20|80|100`
- **텍스트 변경**: `externalCurrentStep.value = '검색 중...'`

### 성공/실패 처리
- **자동 탭 전환**: `activeTab.value = 'internal'`
- **문서 로드**: `editor.content = response.generated_doc_data.content`
- **편집 모드 활성화**: `editMode.value = true`

## 📱 반응형 동작

### 데스크톱 (1024px+)
- **3단 레이아웃**: 사이드바 + 메인 + 상세
- **전체 너비**: `max-w-7xl`
- **사이드바 고정**: `w-64 flex-shrink-0`

### 태블릿 (768px-1023px)
- **2단 레이아웃**: 사이드바 + 메인
- **사이드바 축소**: `w-48`
- **반응형 패딩**: `sm:px-6`

### 모바일 (<768px)
- **1단 레이아웃**: 세로 스택
- **사이드바 접기**: 토글 가능
- **전체 너비**: `w-full`
- **패딩 축소**: `px-4`
